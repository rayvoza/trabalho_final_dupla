# trabalho_final_dupla

Análise lexical e estatística do conto A Cartomante, de Machado de Assis

Descrição:
Este projeto foi desenvolvido para a disciplina Ferramentas Computacionais 
de Modelagem, lecionada por Thomas Nogueira Vilches no segundo semestre de 
2025 como uma disciplina transversal disponível no Portfólio da Unesp.O 
objetivo é realizar uma análise exploratória de dados textuais no R, 
utilizando técnicas de manipulação, limpeza, estatísticas descritivas 
e visualização de dados.

Corpus:
O corpus escolhido é o conto A Cartomante, de Machado de Assis, por ser um 
texto literário clássico e adequado para análises quantitativas 
dentro da área da Linguística.

Objetivos do Projeto:
- Leitura e pré-processamento do corpus Contagem de palavras, frases e linhas;
- Limpeza do texto;
- Criação de tabelas de frequência;
- Construção de pelo menos dois gráficos diferentes;
- Cálculo de medidas estatísticas simples (média, mediana etc.) relacionadas 
à distribuição das palavras;
- Uso de Git e GitHub para desenvolvimento colaborativo.
